# Combine Administrator Simulator — Lore Accuracy Upgrade

Ce pack remplace `src/App.tsx` et `src/index.css` par une version autonome plus strictement Half-Life / Half-Life 2 / Half-Life: Alyx :

- City configurable.
- Scénarios pré-HL2, occupation standard, quarantaine, après Nova Prospekt, uprising.
- Profils de gouvernance : loyaliste, technocrate, tyran, collaborateur, sympathisant secret, gestionnaire quarantaine.
- Système de secteurs avec rébellion, Xen, surveillance, infrastructure, loyauté et peur.
- Unités Combine : Civil Protection, Scanners, Manhacks, Grunts, Soldiers, Ordinals, Suppressors, Elites, Hunters, Striders, Dropships, Gunships, équipes de quarantaine, Advisors.
- Crises narratives avec choix et conséquences.
- Propagande Breencast.
- Directives Citadel.
- Fin de journée simulée.
- Conditions de fin : insurrection, quarantine zone totale, admin remplacé, ville modèle, terreur stérile, humanité préservée.

## Installation

Copie le dossier `src/` de ce pack à la racine de ton repo en remplaçant les fichiers existants :

```bash
cp -r src/* /chemin/vers/combine-administrator-simulator/src/
npm install
npm run dev
```

Ou copie manuellement :

- `src/App.tsx` → `combine-administrator-simulator/src/App.tsx`
- `src/index.css` → `combine-administrator-simulator/src/index.css`
